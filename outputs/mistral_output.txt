Task 3: Code Generation
Prompt:
Write a Python function that checks if a number is prime.

Response:
def is_prime(n):
    if n <= 1:
        return False
    for i in range(2, int(n ** 0.5) + 1):
        if n % i == 0:
            return False
    return True
